kima
About

kima is the name of a really good drink produced in Azores
And now, it’s also a package for the analysis of radial velocity (RV) data.

At its core, kima fits Keplerian curves to a set of RV measurements, using the Diffusive Nested Sampling algorithm1 to sample the posterior distribution for the model parameters. Additionally, the code can calculate the fully marginalized likelihood (or evidence, ZZZ) of a model with a given number of Keplerians and also infer the number (NpN_pNp​) of Keplerian signals detected in a given dataset. In this case, NpN_pNp​ is a free parameter in the fit and its posterior distribution is estimated2, 3.

The code is under active development at this GitHub repository.
This page hosts the documentation and examples, as well as the latest news.
